Hexo搭建博客

发表于 2017-12-04 |

概述

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

碰到的问题

网上已经有许多相关的入门教程，推荐查看官方文档。我在使用Hexo搭建博客的过程中，遇到了一些问题，记录如下：

  • 在另一台电脑如何同步Hexo站点配置，同步博客内容

解决办法

初次使用GitHub、Hexo：

  1. 安装Node.js、Git
  2. git添加ssh key
  3. 创建仓库(名为username.github.io)，设置Github Pages，创建两个分支source，master
    • source：这个分支保存站点相关配置，设置为默认分支
    • master：这个分支用来发布博客静态页面(User and Organization Pages只能通过master分支发布)
  4. 安装hexo：npm install hexo-cli -g
  5. 初始化配置文件、扩展包：hexo init xxx（创建一个名为xxx的Hexo站点文件夹，并进行初始化配置）
  6. 进入Hexo站点文件夹：cd xxx
  7. 下载扩展包（与package.json同步）：npm install
  8. 安装其它Hexo扩展包
    • 使用hexo d来部署到git上，npm install hexo-deployer-git --save
    • 建立RSS订阅，npm install hexo-generator-feed --save
    • 建立站点地图，npm install hexo-generator-sitemap --save
    • 本地搜索（Next主题），npm install hexo-generator-searchdb --save

  9. 进行站点配置，打开_config.yml文件

  10. 清除缓存：hexo clean

  11. 生成文件：hexo g
  12. 本地预览：hexo s
  13. 部署到git：hexo d

如果换了电脑：（不需要hexo init，初始化配置会覆盖掉之前的站点配置，扩展包）
